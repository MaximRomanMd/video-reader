# Promotion kit

Ready-to-post drafts. **You publish these** under your own accounts — nothing here is auto-posted.
Work top-down; the demo GIF (in `../../assets/demo.gif`) is the single biggest conversion lever, so
record a real one before the big pushes.

| File | Channel | When |
|---|---|---|
| `show-hn.md` | Hacker News (Show HN) | after the GIF + README are polished |
| `reddit.md` | r/ClaudeAI, r/ClaudeCode, r/commandline | same day as Show HN or the day after |
| `twitter-x.md` | X / Twitter thread | pin it; reply to Anthropic/Claude Code posts |
| `devto.md` | dev.to / Hashnode / personal blog | the deep-dive that everything else links to |
| `awesome-list-pr.md` | PRs to awesome-claude-code lists | evergreen — do once, stays forever |
| `discord.md` | Anthropic / Claude Developers Discord | share in show-and-tell / skills channels |

### Suggested order
1. Land the **demo GIF** + badges (done in README).
2. Publish the **dev.to** deep-dive (gives everything a URL to point at).
3. **Show HN** + **Reddit** + **X thread** on the same day (cross-momentum).
4. Open the **awesome-list PRs** (evergreen discovery).
5. Drop it in **Discord**.

### Timing tips
- Show HN: weekday, ~8–10am US Eastern, is generally a good window.
- Reply to your own Show HN early with the "why I built it" context (don't put it in the post body).
- Engage with every early comment — HN/Reddit reward author responsiveness.
